# KleurIT - Ingeniería de Software Futurista

KleurIT es una landing page de alto rendimiento diseñada para agencias de desarrollo de software premium. Implementa el sistema de diseño **Obsidian Nexus**, enfocado en una estética cinematográfica, futurista y de alta precisión técnica.

## Características
- 🚀 **Diseño Premium**: Estética dark-mode con efectos de Glassmorphism y Edge Lighting.
- 📱 **Totalmente Responsivo**: Adaptado para dispositivos móviles, tablets y escritorio.
- ✉️ **Formulario Funcional**: Integración directa con Formspree para recepción de leads en Gmail.
- ✨ **Micro-interacciones**: Animaciones de revelación al hacer scroll y efectos de hover con resplandor neón.
- 👨‍💻 **Efecto de Terminal**: Consola interactiva en tiempo real en la sección Hero.

## Tecnologías Utilizadas
- HTML5 Semántico
- CSS3 (Vanilla) con variables y Grid/Flexbox
- JavaScript moderno (ES6+)
- [Formspree](https://formspree.io/) para el backend del formulario

## Instalación
1. Clona este repositorio.
2. Abre `index.html` en tu navegador o usa una extensión como Live Server en VS Code.
